# Betty

### 1. Betty(Better than yesterday)?
어제보다 더 나은 오늘을 위해
오늘의 일정과 날씨, 체크 사항, 준비물품 등을 알려줌으로 더 나은 하루를 보낼 수 있도록 도와주는 어플리케이션.



### 2. Information
- Java(language), Android Studio(tool), Firebase(DB)
- 공공데이터 동네예보 API, NFC API 사용
- Google ID 연동
- GPS, Push message, voice 기능
